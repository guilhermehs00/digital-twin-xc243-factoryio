## Dicas para boas capturas

- Use resolução mínima de 1280×720 px.
- Salve em formato **PNG** (melhor qualidade que JPG para capturas de tela).
- Comprima imagens grandes (>1 MB) usando ferramentas como **TinyPNG** ou **Squoosh** antes de subir, para evitar repositório pesado.
- Mantenha os mesmos **nomes de arquivo** listados acima, ou ajuste as referências no README.
